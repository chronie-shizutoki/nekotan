# 静時ねこたんの日記帳🐾

にゃんこみたいに優しく、毎日の思い出をかわいく記録できる日記アプリだにゃ～♡

✨ 可愛いインターフェースで、お気持ちを大切に保管するよ ✨

## 国際化にゃ～

本アプリケーションは、とってもかわいい多言語対応を目指しているにゃ～♡

- [英語](README-en.md)
- [韓国語](README-ko.md)
- [中国語（簡体字）](README-zh.md)
- [中国語（繁体字）](README-tw.md)

## 本地化リソース

本アプリケーションで使用している外部リソースは全て本地化されています：

### フォント

`public/vendor/fonts` ディレクトリに格納：

### JavaScriptライブラリ

## にゃんとこ特徴✨

- 💕 にゃんなインターフェースで、触るだけでハートが踊る超使いやすさにゃ～💖
- 📝 かわいいカテゴリー分類機能で、お気持ちをきれいに整理するのにゃん♪📂

- 📊 毎日自動でバックアップしてくれる安心機能、寝ても安心にゃ～😴💤
- 📱 モバイルでも指先でかわいく操作できるよ～📱💕

## 動かすために必要なもの🍼

- Node.js >= 14.0.0 (にゃんのバージョン以上だにゃ～)

## 協力するにゃ～方法🤝

静時ねこたんをもっとかわいく、もっと便利にするお手伝いを募集するにゃ～🙏

1. リポジトリをにゃんとフォークして、自分のお家にコピーするにゃ～🍴
2. 新しいブランチを「feature/かわいい機能」みたいにかわいく作るにゃ～🌱
3. 変更をコミットして、メッセージもかわいく書いてプッシュするにゃ～📤
4. Pull Requestを作成して、変更内容を詳しく教えて送るにゃ～✉️

一緒に静時ねこたんをもっとかわいく成長させようにゃ～！💕 プルリクエストを送ってくれたら、お返事を超～早くするよ～✧*｡٩(ˊᗜˋ*)و✧*｡ ご協力ありがとうにゃ～♡
```bash
# リポジトリをにゃんとクローンするにゃ～
git clone https://github.com/quiettimejsg/nekotan.git

# にゃんとこ依存パッケージを入れるにゃ～
```

## かわいい環境設定✨

`.env`ファイルを作成して、お部屋の設定をかわいくカスタマイズするのにゃ～🔧:

```env
# ポート番号設定（変更しないでね～）🔒
PORT=3000
# 環境設定（本番/開発）🌳
NODE_ENV=production
# 最大ファイルサイズ（5MB）📦
MAX_FILE_SIZE=5242880
# ログレベル（情報量調整）📝
LOG_LEVEL=info
# バックアップ保持期間（30日）⏳
BACKUP_RETENTION_DAYS=30
# CORS設定（誰からでもアクセス可）🌐
CORS_ORIGIN=*
```

## 起動方法

開発環境でアプリを元気に起動するのにゃ～💻✨:
```bash
npm run dev
```

本番環境でしっかり稼働させるのにゃ～🚀:
```bash
npm run prod
```

その他のコマンド:
- `npm run stop`: アプリをにゃんと停止するにゃ～🛑
- `npm run restart`: アプリを元気に再起動するにゃ～🔄
- `npm run logs`: ログを見て様子をチェックするにゃ～🔍

## かわいく使う方法～♡

1. アプリを起動して、にゃんと愛でるようにログインするのにゃ～🔑✨
2. ピンクの「新しい日記」ボタンをキュンッとクリックするのにゃ～✏️💖
3. 日記のタイトルとお気持ちをかわいく入力するのにゃ～📝💭
4. カテゴリーとタグを選んで、おしゃれに整理するのにゃ～🏷️🎀
5. 「保存」ボタンをポチッと押して、大事な思い出をしっかり保管するのにゃ～💾💕

## バックアップについて

- CSVファイルは毎日自動でバックアップされるにゃ～💾
- バックアップファイルは `backups/` フォルダーに大事に保管されるにゃ～📁
- デフォルトで30日間愛でてあげるけど、環境変数で変更できるにゃ～📅

## セキュリティ

- Helmet.jsで頭を守るセキュリティヘッダー設定🎩
- CORS設定で仲良く通信するようにしてるにゃ～🤝
- 入力データをチェックして怪しいものは通さないにゃ～🔍
- エラーが起きても優しく教えてくれるハンドリングシステム💬

## ログたち📖

アプリの日常とお世話記録がここにあるにゃ～:
- アプリケーションログ: `logs/output.log` → アプリの一日記録帳✨
- エラーログ: `logs/error.log` → トラブル時のお助けメモ📝
- アクセスログ: `logs/access.log` → いつ来てくれたかのお客様帳簿👀
